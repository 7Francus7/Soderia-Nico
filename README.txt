SODERIA NICO - SISTEMA DE GESTION v2.0
=======================================

Este sistema ha sido migrado COMPLETAMENTE a TypeScript utilizando el framework Next.js.
Se ha eliminado Python y el backend separado para mejorar la velocidad, seguridad e instalación.

CARACTERISTICAS:
- Interface Premium de alta velocidad.
- Base de Datos SQLite integrada con Prisma.
- Autenticación segura con NextAuth.
- Diseño responsivo para tablets de choferes y PC de administración.

COMO INICIAR:
1. Ejecuta el archivo "INICIAR_SISTEMA.bat".
2. El sistema se abrirá automáticamente en tu navegador.
